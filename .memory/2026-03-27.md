## URL 参数路由 - 表演模式直达

- 新增 URL 参数路由功能，支持从外部直接跳转到指定模式
- 参数说明：
  - `?mode=perform` 或 `?mode=performance` → 直接进入表演模式
  - `?mode=practice` → 直接进入练习模式
  - `?role=duck1~duck6` → 预选角色
  - 可组合使用：`?mode=perform&role=duck3`
- 实现方式：在 `init()` 完成后检测 URL 参数，等待资源加载完成后自动切换模式
- 表演模式直达链接：`https://claw-abx.pages.dev/?mode=perform`

## 队形编排改为 2-2-合 模式

- 修改文件：`js/lyrics-data.js`
- 原模式 "2-1-合"：每段前2句由2人领唱 → 后2句由1人独唱 → 合唱
