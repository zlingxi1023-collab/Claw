
## hls-loader.js v6 修复

- **核心 bug**：iOS 路径 `finishIOSLoading` 提前设置 `loadingComplete = true`，导致 `finishLoadingCommon` 的 `if(loadingComplete) return;` 直接退出，用户永远卡在"正在初始化播放器"
- **修复**：移除 `finishIOSLoading` 中的提前设置，统一由 `finishLoadingCommon` 管理
- **新增后台静默预缓存**：iOS 进入应用后，用 Range 请求下载 MP4 前 2MB（moov+首帧），存入 Cache API，加速后续播放和 seek
- GitHub Pages 构建状态已确认 built
